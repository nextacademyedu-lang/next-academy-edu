# Wireframe: Register (`/register`)

## Form Layout
- **Header:** "Create an Account"
- **OAuth:** `Continue with Google`
- **Divider:** "Or"
- **Inputs:** 
  - First Name (50%) | Last Name (50%)
  - Email (100%)
  - Phone Number (100%)
  - Password (100%)
- **Validation Block:** Shows realtime checklist for 8+ chars, uppercase, lowercase, number, special char.
- **CTA:** `Sign Up`.
- **Footer:** "Already have an account? Log in".
