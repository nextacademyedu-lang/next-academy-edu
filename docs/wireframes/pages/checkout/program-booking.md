# Wireframe: Book Round (`/book/:roundId`)

## Split UX
### Right (Summary)
- Fixed sticky card.
- Program details, original price, round dates.

### Left (Setup)
- Pre-filled User details.
- Select Payment Type: full or Installments.
- If Installments: Radio group showing available plans (e.g. 3 Months at $100/mo).
- CTA: `Proceed to Checkout`.
