# Wireframe: Program Details (`/programs/:slug`)

## 1. Hero Section (Split Layout)
- **Left:** Title, Difficulty Level tag, Overall Duration. CTA: `View Upcoming Rounds`.
- **Right:** Main Cover Image or Promo Video.

## 2. Main Content (Left Column - 70%)
- **Overview:** Introduction paragraph.
- **What You'll Learn:** Bulleted list with checkmarks.
- **Who Is This For?:** Target audience details.
- **Syllabus:** Accordion list of modules/weeks.
- **Instructor Bio:** Mini profile card of the instructor.

## 3. Sticky Sidebar (Right Column - 30%)
- **Booking Card:**
  - Round Status (e.g., "Enrolling Now").
  - Start Date & End Date.
  - Location/Format.
  - Total Price.
  - Seats Left indicator (e.g., "Only 3 seats left", styled in Red for urgency).
  - CTA 1: `Book Round` (Primary).
  - CTA 2: `Request Installments` (Secondary/Outline).
