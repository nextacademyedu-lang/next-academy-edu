# Wireframe: Workshops Listing (`/workshops`)

## 1. Hero Section
- **Title:** "All Workshops"
- **Search Bar:** Large input field for searching by keyword.

## 2. Main Content (Split Layout)
### Left Sidebar (Filters)
- **Categories:** Checkboxes for niches (Marketing, Sales, Leadership).
- **Price Range:** Slider or checkboxes.
- **Format:** Online, In-Person, Hybrid.

### Right Column (Results)
- **Grid:** Workshop cards (similar to homepage).
- **Sorting:** Dropdown "Sort by: Newest / Price".
- **Pagination:** At the bottom.
