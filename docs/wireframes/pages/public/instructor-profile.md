# Wireframe: Instructor Profile (`/instructors/:slug`)

## 1. Header Profile
- **Cover Image:** Dark textured background.
- **Avatar:** Circular headshot overlapping the cover.
- **Details:** Name, Tagline, Social Icons (LinkedIn, X).

## 2. Main Content (Split)
### Left (Bio & Content)
- **About:** Rich text covering experience and background.
- **Programs Taught:** Grid showing all Next Academy programs by this instructor.

### Right (Consultations)
- **Sticky Box:** "Book 1:1 Consultation with [Name]".
- **Options:** List of available session types (e.g., "60-Min Strategy - $150").
- **CTA:** `See Availability`.
