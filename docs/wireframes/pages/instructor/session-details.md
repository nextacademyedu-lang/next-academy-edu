# Wireframe: Session Details (`/instructor/sessions/:id`)

## Management Tools

- **Header:** Session Info (Program Title, Round #, Date, Time).
- **Attendance:** Table of enrolled users with Present/Absent toggles per user.
- **Materials:** Upload slides (PDF), worksheets, or any supplementary files per session.
- **Action:** `Complete Session` toggle (marks session as done).

> **Note:** Session recordings are managed exclusively by the Admin via Payload CMS (`/admin/sessions`). The instructor does NOT upload or view recordings here.
