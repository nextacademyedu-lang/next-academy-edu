# Wireframe: Admin Companies (`/admin/companies`)

- New Custom Payload Collection view for B2B Management.
- **List View:** Table showing `Company Name`, `Total Employees (Contacts)`, `Total Sales Value`, and `Preferred Categories`.
- **Detail View (Inside a Company):**
  - Read-only list of all user accounts associated with this Company ID.
  - Computed field displaying total LTV (Lifetime Value) combined from all connected users.
  - Tagging system showing what course categories this company's employees buy the most.
