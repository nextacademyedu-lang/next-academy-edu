# Wireframe: Payments List (`/dashboard/payments`)

## Tabbed View
- **Tab 1: Active Installment Plans**
  - Accordion showing the overall plan (e.g., 3-Month Plan).
  - Inside: List of installments, dates, amounts.
  - Due items have a `Pay Now` red/orange button.
- **Tab 2: Payment History**
  - Table of all successful payments.
  - columns: Date, Ref ID, Amount, Link to Receipt.
