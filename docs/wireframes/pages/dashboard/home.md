# Wireframe: Dashboard Home (`/dashboard`)

## Global Layout
- **Sidebar:** Links to Dashboard, Bookings, Payments, Requests, Profile.
- **Header:** Breadcrumbs, Theme Toggle, Notifications, User Menu.

## View Content
- **Greeting:** "Good Morning, [Name]".
- **Top Metrics (3 Cards):** 
  - Next Session Date & `Join Zoom`.
  - Pending Payments & `Pay Now`.
  - Total Courses Completed.
- **Activity Feed:** List of recent automated alerts.
- **Suggested Next:** Grid of 2 recommended programs based on interests.
