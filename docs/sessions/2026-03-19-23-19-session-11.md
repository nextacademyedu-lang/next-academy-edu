## Completed Tasks

- Updated `programs` schema (additive only):
  - Added `featuredPriority` for manual ordering in featured cards.
  - Added `learnersCount` as aggregate-ready read-only metric field.
- Added new API: `GET /api/home/stats`
  - Computes values from DB collections:

  - Builds normalized frontend card DTO with:
    - title/type/category/instructor
    - enrolled count
    - rating + rating count
    - date/schedule + price + image + href
- Migrated home sections to DB APIs:
  - `src/components/sections/stats.tsx` now fetches `/api/home/stats`.
  - `src/components/sections/featured.tsx` now fetches `/api/home/featured-programs`.
- Fixed upcoming events contract mismatch:
  - `src/app/api/upcoming-events/route.ts` now returns flat event shape expected by `UpcomingEvents` UI.
- Added i18n keys required by dynamic featured cards:
  - `Featured.subtitle`, `Featured.enrolled`, `Featured.noRating` in `en.json` and `ar.json`.
- Updated logs/docs:
