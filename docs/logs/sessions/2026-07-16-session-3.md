## 🏗️ Docker Architecture

```
nginx (80/443) ──► app:3001 ──► postgres:5432
                           └──► redis:6379
```

- postgres + redis + app: internal network فقط (مش exposed)
- nginx: الوحيد اللي بيستقبل traffic من الخارج
- certbot: profile منفصل للـ SSL setup
